Detailed Description

Provides an interface for implementing a GLFW-based application.

The base class owns the window, frame timer (clock), and window properties while inheritors can manage additional resources like cameras as well as OpenGL objects in the dedicated virtual methods.

For example usage, see https://github.com/jakerieger/ParticleSim/blob/64638194ccbd87ce6ab447fbc6c3a3d4797739fb/Source/main.cpp#L15
